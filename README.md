# Slickdeals Test Case Project

## Overview
This repository contains the manual test cases for the Slickdeals website. The tests are focused on ensuring that all functionalities of the website perform as expected across different browsers and scenarios.

## Test Case Details
The test cases cover various functionalities of the Slickdeals website, including:
- **Log In / Sign Up**: Ensures that users can log in and sign up successfully.
- **Browser Compatibility**: Tests the site's across different browsers.
